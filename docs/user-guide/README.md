EasyLoupe is organized around a simple culling workflow: open a folder, review
photos, tag the strongest frames, compare similar images, then organize or
export metadata for the next app in your editing process.

The app has several views, but you do not need to learn them all at once. Start
with culling view, use browse view when you want to scan a whole folder, and
use compare view when a small set of photos needs closer inspection.

## 1. Guides

- [Getting Started](getting-started.md): launch EasyLoupe, open folders or
  files, choose whether to include subfolders, and move from photo viewer mode
  into the full culling workspace.
- [Culling View](culling-view.md): use the main workspace, thumbnail strip,
  viewer, top bar, sort controls, and filter button.
- [Browse View](browse-view.md): scan the whole folder as a grid, make
  selections, and return to culling view.
- [Compare View](compare-view.md): inspect selected photos side by side, choose
  the active pane, adjust the compare limit, and return to browse or culling
  view.
- [Tagging And Filtering](tagging-and-filtering.md): apply ratings, color
  labels, picked/rejected flags, undo metadata edits, and hide photos with
  session-only filters.
- [Scenes](scenes.md): detect photo sequences, work with scene stacks, navigate
  the scene strip, and manually merge or break scenes.
- [Inspection Tools](inspection-tools.md): use fit/manual zoom, split view, AF
  point display, EXIF and histogram details, clipping warnings, and visible
  region minimaps.
- [Organize And XMP](organize-and-xmp.md): reorganize photos into folders,
  write shared XMP sidecars, choose conflict behavior, and undo completed file
  operations.

## 2. A Typical Pass

1. Open a folder and let EasyLoupe build previews.
2. Review the shoot in culling view or press `G` for browse view.
3. Select strong candidates and assign ratings, labels, or picked/rejected
   flags.
4. Press `C` to compare similar selected photos side by side.
5. Run scene detection when a shoot contains bursts or repeated sequences.
6. Use `Organize Photos` to copy/move files by metadata or write XMP sidecars.

Press `?` or choose `Help > Keyboard Shortcuts` in the app to see shortcuts for
the current view.
